Sponsors

We go to great lengths to run a focused, welcoming event. That includes providing a sponsored lunch for attendees and letting attendees know that they are part of a large, welcoming professional Python community. We also run wrap-up events so that organizers can discuss what to improve next time.

This takes time, space, and money. We're grateful to our sponsors so far who have made this possible.
